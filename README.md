# Charliemania Web Oficial Provisional

Web madre de Charliemania: nutrición, fuerza y libertad metabólica.

## Cómo subir a GitHub

1. Descomprime este ZIP.
2. Crea un repositorio en GitHub llamado `charliemania-web`.
3. Sube todos los archivos de esta carpeta al repositorio.
4. Haz commit con el mensaje: `Primera versión web Charliemania`.

## Cómo publicar en Vercel

1. Entra a Vercel.
2. Add New → Project.
3. Importa el repositorio `charliemania-web`.
4. Vercel debería detectar Vite automáticamente.
5. Configuración:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
6. Deploy.
